# 📖 Simulador de Leitura de Livro
Este projeto utiliza Programação Orientada a Objetos (POO) em Python para simular a experiência de ler um livro. O programa controla o progresso das páginas, exibe animações de carregamento no terminal e identifica quando o leitor chega ao fim da obra.

# 🛠️ Funcionalidades
- Criação de Objetos: Utiliza a classe Livro para instanciar obras com títulos e quantidades de páginas específicas.
- Avanço Dinâmico: O método avancar_paginas(n) permite que o usuário "folheie" o livro, avançando um número determinado de páginas por vez.
- Interface Enriquecida: Uso da biblioteca rich para exibir textos coloridos e emojis diretamente no terminal, tornando a experiência visualmente atraente.
- Simulação de Tempo: Implementação de sleep para criar um intervalo realista entre a passagem de cada página.
- Verificação de Conclusão: O sistema monitora o contador de páginas e emite um aviso especial quando o livro é finalizado.
